# Docker ile PHP Projesi

Bu proje, Docker üzerinde çalışan basit bir PHP uygulamasıdır.

## Gereksinimler

- Docker
- Docker Compose

## Kurulum ve Çalıştırma

1.  **Projeyi bilgisayarınıza indirin (clone).**

2.  **Aşağıdaki komutu çalıştırın:**
    ```bash
    docker-compose up --build -d
    ```
